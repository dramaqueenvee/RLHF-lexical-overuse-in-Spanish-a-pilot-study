# RLHF-lexical-overuse-in-Spanish-a-pilot-study
This project replicates the procedure of Juzek & Ward (2025) to investigate  whether Reinforcement Learning from Human Feedback (RLHF) causes lexical overuse in  Spanish-language LLMs, using Google's Gemma 2 2B model family.

## Dependencies

### Python (local)
- biopython
- langdetect
- pandas
- scipy
- numpy

Install with:
pip install biopython langdetect pandas scipy numpy

### Google Colab
- transformers
- torch
- huggingface_hub

### Models
- google/gemma-2-2b (HuggingFace — requires license acceptance)
- google/gemma-2-2b-it (HuggingFace — requires license acceptance)
Both available at https://huggingface.co/google
